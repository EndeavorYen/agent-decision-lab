# Agent Decision Lab v0.1.0

This is the first GitHub Release for Agent Decision Lab.

## Release Scope

- Local-first file-backed experiment metadata.
- Git branch and worktree isolation.
- Decision points, savepoints, variants, strategies, artifacts, evaluations,
  comparisons, and guidance drafts.
- `adl case-study` workflow commands.
- Qualitative `--no-score` comparison with no forced winner.
- Worktree lifecycle inspection and dry-run cleanup.
- Redacted JSON, Markdown, Mermaid, SVG, and dashboard-style HTML exports.
- Provider-neutral `adl adapter` and `adl plugin` recipes.
- `adl doctor` production readiness checks.
- Production onboarding documentation.
- Sanitized Reality Slap Skill case-study documentation.

## No npm publish

This release creates a Git tag and GitHub Release only. It does not publish the
package to npm.

## Verification

Release verification command:

```bash
npm run verify
```

The release gate runs:

- `npm test`;
- `npm run smoke`;
- `git diff --check`;
- `npm pack --dry-run`.

GitHub Actions runs the same verification command on `main`.

## Live Evidence

The v0.1.0 release is supported by private live case studies that exercised:

- multi-agent and multi-worktree development;
- productized case-study commands;
- qualitative no-score evaluation;
- redacted dashboard exports;
- worktree lifecycle commands;
- a public Reality Slap Skill target checkout with private `.agent-lab/`
  experiment state.

Raw live transcripts, private artifacts, and target-local paths are not included
in the open-source repository.

## Release Procedure

1. Run `npm run verify`.
2. Confirm `git status --short --branch` is clean.
3. Push `main`.
4. Create annotated tag `v0.1.0`.
5. Push the tag.
6. Create the GitHub Release from this file.
7. Do not run `npm publish`.
